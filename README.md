# PSoC 101 Tutorial Code for CY8CPROTO-063-BLE

This repository contains the **PSoC 101 tutorial code originally provided by Cypress**, adapted for the **CY8CPROTO-063-BLE** board.

## Overview

The goal of this project is to make the PSoC 101 tutorial examples work on the **CY8CPROTO-063-BLE** board.  
Since many tutorial examples are often shown with other boards, this repository includes the necessary adjustments for this specific hardware platform.

## Hardware

- CY8CPROTO-063-BLE
- Onboard LEDs
- Onboard buttons / switches

## Software

- PSoC Creator
- Required device support for CY8CPROTO-063-BLE

## Project Structure

This repository currently includes:

- `PSoC101.cywrk`
- `Lessons1.cydsn`
- `Lessons2.cydsn`

## Notes

The original tutorial code was created by **Cypress** and is now part of the **Infineon** ecosystem.  
This repository is an adapted version intended for learning, testing, and experimenting with the **CY8CPROTO-063-BLE** board.

## Adaptations

The main changes compared to the original tutorial examples include:

- board adaptation for **CY8CPROTO-063-BLE**
- adjusted pin configuration
- usage of onboard LEDs and switches
- small logic changes where needed for compatibility

## Purpose

This project is mainly intended as a learning resource for working through the **PSoC 101** tutorials on the **CY8CPROTO-063-BLE** board.

## Disclaimer

The original tutorial concept and example code are based on material from Cypress / Infineon.
